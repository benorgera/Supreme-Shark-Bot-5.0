Hate resale pricing?
This is for you.

Supreme Shark Bot is a highly advanced auto-checkout software for Supreme New York. Operating through Mozilla Firefox and driven by Selenium, it enables you to checkout with up to three items at a time using keywords and has multiple features to confirm the right item is purchased, every time. With an average checkout time of 2-3 seconds, Supreme Shark Bot will ensure that you successfully checkout.



Want to see how it works?

Supreme Shark Bot has muiltiple features to ensure it purchases the right item everytime, and lightning fast.

  • Keyword Searching finds the item link using an efficient algorithm.
  • Unsure of Color enables you to enter potential and incorrect color keywords, for items whose colors' you're not sure of, seeing as colors aren't released in the lookbook/preview.
  • Disambiguation prompts you asking which item is correct when two or more items have the same number of keywords.
  • Multiple instances enables you to get more than one of the same item on one computer, by using different checkout info for each item.
  • Early Link support waits for an early link to become active, and then adds to cart from there.
  • Sold Out support skips attempting to add sold out items when the user is purchasing more than one item.
  • Updates are released at least once a month and are free and automatic, if you run the bot with an outdated version it will automatically update itself.
  • Auto Process Payment lets you to choose if the bot should complete checkout, allowing you to do so manually.
  • Disable Images enables you to load Supreme without images, making the bot run much faster.
  • Saving/Loading profiles enable you to save checkout info to be loaded during future uses of the bot. These files are encrypted so that no one can get at your checkout info.
  • Double Checking scans every field of the bot you fill out, making sure that there are no discrepancies which could prevent a successful checkout.
  • No Color support enables you to checkout with items which are listed with no color (this is used mainly for accessories).
